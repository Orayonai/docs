## Configurações globais

As configurações são definidas no arquivo `docs.json` na raiz do projeto.

### Cores

```json
{
  "colors": {
    "primary": "#5636d1",
    "light": "#7C5CE7",
    "dark": "#4527A0"
  }
}
```

### Logo

```json
{
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg"
  }
}
```

### Favicon

```json
{
  "favicon": "/favicon.ico"
}
```

## Configurações do workspace

Acesse **Configurações** no painel para:

- Gerenciar integrações
- Configurar webhooks
- Definir permissões de usuários
- Personalizar notificações
