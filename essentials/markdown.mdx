## Títulos

```markdown
# Título 1
## Título 2
### Título 3
```

## Formatação de texto

| Sintaxe | Resultado |
|---------|-----------|
| `**negrito**` | **negrito** |
| `*itálico*` | *itálico* |
| `~~riscado~~` | ~~riscado~~ |
| `` `código` `` | `código` |

## Listas

```markdown
- Item 1
- Item 2
  - Sub-item

1. Primeiro
2. Segundo
```

## Links e imagens

```markdown
[Texto do link](https://url.com)
![Alt da imagem](/caminho/imagem.png)
```

## Citações

```markdown
> Esta é uma citação
```
