---
title: 'Obter dados'
api: 'GET /v1/messages/{id}'
description: 'Recupera informações de uma mensagem específica'
---

## Parâmetros

<ParamField path="id" type="string" required>
  ID único da mensagem
</ParamField>

## Resposta

<ResponseField name="id" type="string">
  Identificador único da mensagem
</ResponseField>

<ResponseField name="content" type="string">
  Conteúdo da mensagem
</ResponseField>

<ResponseField name="status" type="string">
  Status: `pending`, `sent`, `delivered`, `read`
</ResponseField>

<ResponseField name="created_at" type="string">
  Data de criação (ISO 8601)
</ResponseField>

## Exemplo

<CodeGroup>

```bash cURL
curl -X GET https://api.orayon.ai/v1/messages/msg_123 \
  -H "Authorization: Bearer SEU_TOKEN"
```

```javascript Node.js
const message = await orayon.messages.get('msg_123');
console.log(message);
```

</CodeGroup>

## Resposta de sucesso

```json
{
  "success": true,
  "data": {
    "id": "msg_123",
    "content": "Olá!",
    "status": "delivered",
    "created_at": "2025-01-15T10:30:00Z"
  }
}
```
