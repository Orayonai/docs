## Sobre o Windsurf

Windsurf é um IDE desenvolvido pela Codeium com IA integrada.

## Configuração

1. Abra o Windsurf
2. Acesse configurações de documentação
3. Adicione a URL: `https://docs.orayon.com.br`

## Uso

Use o chat do Cascade para perguntar sobre a Orayon:

```
Como configuro automações na Orayon?
```

## Recursos

- Autocompletar com contexto da documentação
- Chat com conhecimento da API Orayon
- Sugestões de código baseadas em exemplos
