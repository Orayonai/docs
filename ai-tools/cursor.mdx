## Sobre o Cursor

O Cursor é um IDE com IA integrada que pode acessar sua documentação Orayon.

## Configuração

1. Abra o Cursor
2. Pressione `Cmd/Ctrl + L` para abrir o chat
3. Use `@Docs` e selecione a documentação Orayon

## Uso

```
@Docs Como configuro um webhook na Orayon?
```

O Cursor vai buscar informações relevantes na documentação e responder com contexto.

## Benefícios

- Acesso rápido à documentação sem sair do IDE
- Respostas contextualizadas para seu código
- Integração nativa com IA
